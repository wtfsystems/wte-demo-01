### Simple demo game using WTEngine.

Requires the following libraries to build:
- [WTEngine](https://github.com/wtfsystems/wtengine)
- [Allegro](https://github.com/liballeg/allegro5)

---

To build, run from here:
```
cmake .
make
```

---

### Controls:
 - W A S D - move
 - Right Alt - shield
 - Right Ctrl - laser
 - ESC - pause
